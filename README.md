# 🔥 PulseAI — News & Markets Dashboard

A sleek, real-time news aggregator with AI-powered insights and stock market tracker. Built as a pure static web app — deploy on GitHub Pages in seconds.

![PulseAI Screenshot](https://via.placeholder.com/800x400/0a0b0f/4f7cff?text=PulseAI+Dashboard)

## ✨ Features

- **📰 Live News Feed** — Pulls top headlines from NewsAPI, auto-classified into topics by Gemini AI
- **✦ Ask Gemini** — Ask any question about today's news; Gemini answers with context
- **✦ AI Article Summaries** — Click any article for a Gemini-powered summary + key insight
- **📈 Stock Tracker** — Real-time prices via Alpha Vantage with scrolling ticker tape
- **🤖 AI Stock Recommendations** — BUY / SELL / HOLD ratings powered by Gemini for every stock
- **⭐ Watchlist** — Add/remove favorites, persisted in localStorage
- **💹 AI Market Brief** — On-demand market overview generated by Gemini
- **🔄 Auto-refresh** — News and prices update every 15 minutes

## 🚀 Deploy to GitHub Pages

1. **Fork / clone** this repo
2. Go to **Settings → Pages → Source: Deploy from branch → main → / (root)**
3. Your app is live at `https://yourusername.github.io/pulseai/`

> **No build step required.** Pure HTML/CSS/JS.

## 🔑 API Keys (all free tiers available)

API keys are stored in your browser's `localStorage` — they are **never committed to git**.

| Service | Purpose | Free Tier | Link |
|---------|---------|-----------|------|
| **Google Gemini** | AI features (required) | ✅ Generous free tier | [Get key](https://aistudio.google.com/app/apikey) |
| **NewsAPI** | Live news headlines | ✅ 100 req/day | [Get key](https://newsapi.org/register) |
| **Alpha Vantage** | Stock prices | ✅ 25 req/day | [Get key](https://www.alphavantage.co/support/#api-key) |

> **NewsAPI limitation**: Free tier only works from localhost. For a deployed site, you'll need a paid plan or proxy. The app gracefully falls back to demo data if the key is absent.

### Setting Your Keys

Click the **⚙ gear icon** in the top-right corner and paste your API keys. They're saved to `localStorage` immediately.

## 🛠 Local Development

```bash
# Clone the repo
git clone https://github.com/yourusername/pulseai.git
cd pulseai

# Serve locally (required for NewsAPI free tier)
npx serve .
# or
python3 -m http.server 8080
```

Then open `http://localhost:8080` and add your API keys via ⚙.

## ⚙️ Customization

**Change default stocks** — Edit the `popular` array in `js/stocks.js`:
```js
popular: ['AAPL', 'MSFT', 'GOOGL', 'NVDA', 'TSLA', ...],
```

**Change news topics** — Edit the `topic-bar` buttons in `index.html`.

**Adjust AI prompts** — All Gemini prompts are in `js/gemini.js`.

**Theme colors** — All colors are CSS variables in `css/style.css`:
```css
:root {
  --accent: #4f7cff;   /* Primary blue */
  --green:  #2dd4a0;   /* BUY / positive */
  --red:    #ff5a6e;   /* SELL / negative */
  --yellow: #f5c842;   /* HOLD / warning */
}
```

## 🔒 Security Notes

- API keys are stored in `localStorage` only — never in source code
- `.gitignore` has no secrets to protect (there are none in the repo)
- For production use, consider a backend proxy for API keys
